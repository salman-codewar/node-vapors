# Node Vapors Boilerplate

## Tools 
1. Node Js (express)
2. Typescript
3. Graphql Apollo (codegen)
4. ESLint
5. Prettier
4. Husky

## Get Started

1.) Clone repository `https://github.com/salman-codewar/node-vapors.git`

2.) `yarn install` in directory

3.) `yarn develop` to spin up server in development
